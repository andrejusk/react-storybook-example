## Installation

```bash
# if required, run:
#
#  $ npm i -g pnpm
#

pnpm i
```

## Use

Run storybook:

```bash
pnpm storybook
```

Run unit tests:

```bash
pnpm test
```

Run application:

```bash
pnpm dev
```

Build dist:

```bash
pnpm build
```

Run dist:

```bash
pnpm start
```
